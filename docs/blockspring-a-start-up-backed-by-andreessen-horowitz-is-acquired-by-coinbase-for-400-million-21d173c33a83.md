菲亚特至加密交易所[比特币基地](https://www.coinbase.com/)，在一次战略行动中，收购了数据处理初创公司 Blockspring。 [Blockspring](https://www.blockspring.com/) 专门帮助用户将各种商业 API 连接到数据可视化应用程序，如 google sheets 或 Excel。
