# 由销售人员更新的世界上最有力的 25 句名言

> 原文：<https://medium.com/swlh/25-of-the-worlds-most-powerful-quotes-updated-by-a-salesperson-2f5c2b40aac7>

![](img/7f5283be46108684e62b5cb46a429b72.png)

在工作中成为一名优秀的销售人员，就是在家里过着美好的生活。

同样，在战争中，你在训练中流血越多，在战斗中流血就越少。

任何优秀的销售人员都知道，最好的销售不是为了生存，最好的生存是为了销售。因此，任何销售人员都可以写出自己的版本，甚至是世界上一些最有力的名言，这是很自然的。

这个列表是对这些伟大的思想家和各地销售人员的敬意。

# 1."你必须成为你希望在这个世界上看到的变化."—甘地

> "出售你认为会创造你希望看到的世界的产品."

# 2.“每个人都是天才。但是如果你根据一条鱼爬树的能力来判断它，它会一辈子都相信自己是愚蠢的。”——阿尔伯特·爱因斯坦

> 每个人都可以销售，但是如果销售人员的心态不对，他/她就不能最大限度地发挥自己的潜力。掌握一个你天生适合的销售渠道，并加以利用。”

# 3.“犯错误的一生不仅比无所事事的一生更可敬，也更有用。”乔治·伯恩哈德·肖

> “尝试不同的销售渠道对于发现哪种渠道最适合你的产品和市场是必要的。从这一努力中获得的知识、经验和适应力将带来显著的个人和专业成长。”

# 4."害怕自己会受苦的人，已经因为害怕而受苦了."—米歇尔·德·蒙田

> "害怕要求命令的人，已经因为害怕要求而受苦了."

# 5.“我们有时认为贫穷只是饥饿、赤身裸体和无家可归。没有人要，没有人爱，没有人关心的贫穷是最大的贫穷。”—特里萨修女

*“一个销售人员的个人生活和他/她的职业生活一样重要。最好的销售人员不是为了活着而销售，而是为了销售而活着。”*

# 6.“爱是一个动词。爱——感觉——是爱的果实，是动词。”—斯蒂芬·柯维

> 金钱是销售这个动词的副产品。然而，掌握销售技巧需要带着爱去工作，去创造你不直接追求的副产品(金钱)。

# 7.“生活真的很简单，但我们坚持要把它变得复杂。”——孔子

> 销售很简单，有太多的方法来剥销售猫的皮，而我们产品和市场的复杂性使它变得复杂。

# 8.“如果你不喜欢某样东西，就去改变它。如果你不能改变它，那就改变你思考问题的方式。”—玛丽·恩格尔布雷

> *在销售中，拥有正确的心态与拥有正确的技能和工具同样重要。*

# 9."在为别人寻找幸福的过程中，你会在自己身上找到幸福。"—未知

> 销售应该只是让你的客户开心。你会发现你真的关心他们的幸福，他们会回报你的。

# 10.“上帝啊，请赐予我平静去接受我不能改变的事情，赐予我勇气去改变我能改变的事情，并赐予我分辨两者的智慧。”—雷茵霍尔德·尼布尔

> 承认你的优势和劣势，并利用你的优势开发最强的渠道来达到你的市场。

# 11.“昨天是历史，明天是未知，今天是上帝的礼物，这就是为什么我们称它为现在。”—比尔·基恩

> 计划和目标设定是好的，但是专注于下一个电话的纪律将会引导你实现未来的目标。

# 12."生活从来不会因为环境而变得无法忍受，只会因为缺乏意义和目标而变得无法忍受."维克多·弗兰克尔

> “拥抱大客户，培养必要的技能和韧性，让销售成为你目标驱动的生活的一部分。”

# 13.“如果你想要一个小时的快乐——打个盹。如果你想快乐一天，去钓鱼吧。如果你想要一年的幸福——继承一笔财富。如果你想要一生幸福，那就去帮助别人。”—中国谚语

> *“如果你想今天销售成功，就去做客户。如果你想明天销售成功，检查一下那个客户是否被接受了。如果你想要一年的销售成功，那就成为销售经理。如果你想一生都在销售上取得成功，那就去帮助你的销售伙伴。”*

# 14.“当一扇幸福之门关闭时，另一扇门就会打开，但我们经常长时间盯着那扇关闭的门，以至于看不到另一扇门已经为我们打开了。”——海伦·凯勒

> *“当一个推荐渠道关闭时，另一个就会打开。但我们经常在封闭的推荐渠道中停留太久，以至于忘记了看看周围所有的推荐渠道。”*

# 15.“大多数人没有带着理解的意图去听；他们听着，意图回答。”—斯蒂芬·柯维

> *“成功的销售是倾听和提问来理解他们，而不是推销我们自己、产品和公司，这样潜在客户才能理解我们。”*

# 16.“在你诊断自己患有抑郁症或低自尊之前，首先要确保你实际上没有被一群混蛋包围。”——西格蒙德·弗洛伊德”

> “在你加入一个销售组织之前，确保你的经理不是一个混蛋，而且这个团队的文化适合你。”

# 17."挑战让生活变得有趣，战胜挑战让生活变得有意义."——约书亚·马琳”

> 推动我成长的具有挑战性的销售任务让我进入心流状态，心流让生活变得有价值，所以我期待着它们。

# 18."一个被新的经历拉伸的头脑永远不会回到它原来的维度."—奥利佛·文德尔·霍马斯

> 作为一名销售人员和专业人士，思考并找到销售难题的创造性解决方案是成长的最佳途径。

# 19.“我们生存的唯一方式就是成长。我们成长的唯一途径就是改变。我们能改变的唯一方法就是学习。我们能学习的唯一方法就是暴露在外。我们暴露的唯一方式是我们把自己抛到户外。”乔伊贝尔公司

> *卖出不确定，事情不确定的时候生活更精彩。不确定性教会了我适应力，而适应力让我意识到，我需要成为最好的自己，才能卖出更多，成为一名专业人士。*

# 20.“时间对等待的人来说太慢，对恐惧的人来说太快，对悲伤的人来说太长，对欢乐的人来说太短，但对爱的人来说，时间就是永恒。”——亨利·范·戴克

> *带着爱去销售，爱你的潜在客户，这是解决销售人员遇到的最困难的问题和资源限制的方法，因为他们没有足够的时间。*

# 21."我宁愿死得有意义，也不愿活得毫无意义。"科拉松·阿基诺”

> *在销售工作中寻找意义是一种选择，我能从销售或生活中的其他事情中获得多少意义，决定了我的生活能变得有多有意义。*

# 22."成功是从一次失败到另一次失败的磕磕绊绊，但热情不减."——温斯顿·丘吉尔

> 成功的销售是带着前所未有的热情从一个潜在客户到另一个潜在客户，从一个客户到另一个客户。

# 23.“许多人充满热情，但由于他们对自己是谁和能做什么的信念有限，他们从未采取行动来实现他们的梦想”——托尼·罗宾斯

> 或许最重要的技能是挖掘你的激情，并通过你的销售努力来激发它。这就需要采取必要的行动来实现你的销售目标。

# 24."真正的成功是克服对不成功的恐惧."—保罗·斯威尼

> *了解是什么让你成为一名成功的销售人员，将激发你克服任何恐惧并成为最好的自己所需的行动和韧性。*

# 25.“我们生存的唯一方式就是成长。我们成长的唯一途径就是改变。我们能改变的唯一方法就是学习。我们能学习的唯一方法就是暴露在外。我们暴露的唯一方式是我们把自己抛到户外。”乔伊贝尔公司

> 我们销售的唯一方式是打电话。我们获得经验的唯一方法是坚持下去。我们能坚持下去的唯一方法就是学习。我们学习的唯一途径是我们出现。我们每天出现的最好方式是我们破釜沉舟，承诺一生成功销售。
