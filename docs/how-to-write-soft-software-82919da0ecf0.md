# 软件很难

人们通常认为软件很软，容易修改。从某种意义上说，这是对的，如果我们不需要保证我们改变的软件仍然工作。任何有经验的程序员都会告诉你，软件是很难改变的，而且随着时间的推移只会越来越难。
