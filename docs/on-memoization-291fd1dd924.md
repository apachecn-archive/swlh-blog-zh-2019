# 论记忆化

> 原文：<https://medium.com/swlh/on-memoization-291fd1dd924>

**概要:**

使用“开放式”递归:

*   我们可以把记忆逻辑和功能逻辑分开。
*   我们可以在不修改函数语法的情况下记忆函数。
*   我们可以使函数通用于多种记忆类型。
