自由撰稿人不像其他朝九晚五、每周工作五天的员工那样周末休假。自由作家为自己创造休息时间。通常，这是一些小的休息时间，去买食物，出去散散步，跑跑腿，照顾家人，等等。

但是偶尔，你，这个作家，想要休息一天。如果你把自己关起来盯着看，完美的春夏天气将是一种浪费…
