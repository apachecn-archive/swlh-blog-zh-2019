# 你够酷可以成为一名艺术家吗？

> 原文：<https://medium.com/swlh/are-you-cool-enough-to-be-an-artist-f91a5130aea6>

不久前，我和小女儿还在聊高中人气。

“但是妈妈，你在高中的时候并不酷，对吗？”

“哦，我非常酷，”我说。“只是没人知道罢了。”
