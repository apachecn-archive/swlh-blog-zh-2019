分析事件的目的是识别与未来行为的特定结果相关的行为(由事件定义)。本主题跟随营销目标的转变，从根据兴趣或表面趋势对用户进行分组，到根据类似行为对用户进行分组，这在移动开发备忘录上的[博文](https://mobiledevmemo.com/algorithmic-marketing-introduces-a-new-era-of-mobile-ad-targeting/)中有所涉及。
