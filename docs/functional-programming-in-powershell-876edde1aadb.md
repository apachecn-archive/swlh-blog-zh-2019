PowerShell 不是一种纯粹的函数式语言，但是 PowerShell 非常优雅地将一些函数概念集成到它的语义中。通过利用 PowerShell 中的函数式编程概念，您将能够花更多的时间编写干净的惯用 PowerShell，而用更少的时间对您的脚本进行逆向工程。
