# 介绍

[Webtau](https://github.com/testingisdocumenting/webtau) (简称**web**test**au**to mation)是一个用来编写富有表现力和简洁的 REST API 测试的工具和 API。

让我们从一个简单的天气例子开始。您有一个服务器，每当您到达`/weather`终点时，它都会返回当前温度。
