# 写作浪费 5。复杂的简单

将事物复杂化的需求是人类独有的。

当我大约 3 个月前概述这篇文章时，作为 8 篇文章系列的一部分，我为每篇文章创建了一个模板:它们都以一个故事开始，它们都有一个工具或技术，我会画一个简单的图像(用许多颜色)，我会确定进一步的阅读…我知道如果我这样做了，所有的文章都会读得很好，对读者有用…
