# 第一句话就能杀人

> 原文：<https://medium.com/swlh/its-the-first-sentence-that-kills-d71ca8abd669>

自下而上的思考可以帮助你克服写作障碍。

当发现我是做什么的时候，有些人对我说，“哇，你是个作家？你写些什么？”我快速说出我写的各种主题，然后他们告诉我，“我一直想成为一名作家，”或者“我想写一本书(填空)”。我该如何开始？”
