# 你真的需要那个框架吗？

> 原文：<https://medium.com/swlh/do-you-really-need-that-framework-394170ecef7e>

第三方框架和库是(并且可能继续是)一个激烈争论的话题。一些开发人员花更多的时间研究和评估一个潜在的框架，而不是用它开发东西，一些开发人员一遇到问题就去找新的库。本文探讨了使用第三方库和框架的优缺点。
