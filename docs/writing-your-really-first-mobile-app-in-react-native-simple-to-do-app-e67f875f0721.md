好吧，让我们从一个书呆子的介绍开始…🤓。React-Native 是一个框架(由脸书开发)，用一种称为 JavaScript 的 web 开发语言编写 iOS 和 Android 应用程序，并转换为 iOS 和 Android 的跨平台本机应用程序。这样做的好处是，您可以使用您已经习惯的 web 开发语言，并且只需要学习和维护一种语言(JavaScript)。
