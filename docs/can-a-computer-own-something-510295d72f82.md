现在电脑不能做什么？自动化机器和算法现在与我们互动如此之多，很难记得它们不在的时候。

你随身携带的手机就是一台微型电脑。电视也变成了电脑；更不用说你开的车了。我们足不出户就可以通过电脑购物。
