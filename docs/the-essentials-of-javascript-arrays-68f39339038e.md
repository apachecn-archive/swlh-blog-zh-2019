# JavaScript 数组的本质

> 原文：<https://medium.com/swlh/the-essentials-of-javascript-arrays-68f39339038e>

数组是可以保存不同元素或对象的全局对象。在 JavaScript 中，数组也可以用作列表、堆栈或队列。数组可以包含相同数据类型或混合数据类型的项。
