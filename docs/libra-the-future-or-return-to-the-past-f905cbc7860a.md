# 天秤座，未来？或者回到过去

> 原文：<https://medium.com/swlh/libra-the-future-or-return-to-the-past-f905cbc7860a>

脸书揭开新货币天秤的面纱已经有几个星期了。作为这次曝光的一部分，他们已经发布了一份[白皮书](https://libra.org/en-US/white-paper/)和三篇关于[天秤座区块链](https://developers.libra.org/docs/assets/papers/the-libra-blockchain.pdf)、[移动语言](https://developers.libra.org/docs/assets/papers/libra-move-a-language-with-programmable-resources.pdf)和[图书馆](https://developers.libra.org/docs/assets/papers/libra-consensus-state-machine-replication-in-the-libra-blockchain.pdf)的技术论文。尽管 Twitter 立即引起了轩然大波，Github 也进行了一些攻击，但我还是在敲击键盘之前，冒昧地等待并对情况进行了更好的评估。我认为，面对所有的批评，人们不能低估这一*举动*的重要性(是的，这是一个双关语^^).
