如果您使用的是苹果的 macOS，您使用的默认登录 shell 是 Bourne-Again SHell 或简称 BASH。BASH 是一个命令语言解释器，主要用于读取命令和运行其他程序。因此，除非您要更改运行您的操作系统的 shell，否则您在终端中运行的任何命令都将利用 BASH。
