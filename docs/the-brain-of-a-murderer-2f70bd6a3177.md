# 杀人犯的大脑

> 原文：<https://medium.com/swlh/the-brain-of-a-murderer-2f70bd6a3177>

## 美国研究人员最近扫描了 203 名被定罪的杀人犯的大脑，并将其与非杀人犯的大脑进行了比较。他们的发现对未来预防和治疗凶杀案有很大的意义。
