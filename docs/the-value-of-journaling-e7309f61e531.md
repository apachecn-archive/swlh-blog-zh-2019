# 日志的价值

> 原文：<https://medium.com/swlh/the-value-of-journaling-e7309f61e531>

因为你不可能记得所有的事情，有时你需要一个来自你过去的警钟。

我年轻的时候从来不明白使用日志的意义。我认为我不记得的事情并不重要。为什么要把事情写下来？有些事情我无论如何都不想记起——但我应该记起吗？

这些年来，我意识到我的大脑，也许还有你的大脑，屏蔽了某些…
