# 恢复精神

> 原文：<https://medium.com/swlh/refresh-f36e2d8911f4>

UX 案例研究| Luke Jenkins | UXDI Monarchs

现代工作者在办公室花费的时间比以往任何时候都多，以满足对他们不断增长的需求。但是这给工人带来了更大的压力，甚至降低了他们的生产力。研究表明，一天中短暂的休息有助于解决这个问题。问题是——人们真的在休息吗？
