实体框架(EF)核心是微软的跨平台对象关系映射器(orm)。EF Core 提供不同数据存储的抽象，并自动管理数据库资源。EF 核心目标。NET Core，意思是它是跨平台的，可以在 Windows，Mac 和 Linux 上运行。EF 核心不应该与更老的常规实体框架(EF)混淆。NET 框架版本。
